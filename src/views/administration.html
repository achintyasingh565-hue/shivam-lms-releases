Backup & Restore

Your loan records are stored inside this browser. To keep them safe — and to move them between computers — download a backup file regularly.

⤓ Download Backup

Saves all your loan records as a single file. Keep it somewhere safe (pen drive, email, cloud).

⤒ Restore Backup

Load records from a previously downloaded .json backup file. This replaces current data.

Team Members

Add staff accounts so colleagues can sign in with their own username and password on this computer.

Recovery Password Not set

Choose a second password only you know. If you ever forget your login password, type this at the sign-in screen to get back in and set a new one. Nothing is deleted — your records, staff accounts and settings stay exactly as they are. It is stored scrambled on this Mac, so nobody can read it out of the app.

Or generate a one-time code

Generate a one-time recovery code. If the Administrator password is ever forgotten, this code resets it from the sign-in screen — no data loss. Each code can be used once; generate a new one after use.

Auto-lock when idle

If the app is left open and untouched, it locks itself and asks for the password again. Set 0 to turn off.

For the strongest protection, also turn on FileVault (System Settings → Privacy & Security → FileVault) to encrypt your entire Mac. If you forget this password, clear it via your browser/app data — your records stay safe behind your Mac login.
